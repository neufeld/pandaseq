public enum Tagging

The policy for Illumina tags/barcodes in sequence names.


Namespace: Panda
Package: pandaseq-2

Content:

Enum values:

  • PRESENT - The parsing should return an error if the sequence does not have a tag.
  • ABSENT - The parsing should return an error if the sequence has a tag.
  • OPTIONAL - The parsing should not care if the sequence a tag.
