Opener

Description:

public static NextSeq Opener (T user_data, LogProxy logger, out FailAlign? fail)

Create a sequence reader after argument parsing.

This returns a sequence source and a failure handler so that assembly can proceed.

Parameters:

user_data the context
logger The logging proxy to use, if needed.
fail the handler for any sequences which do not align, if desired.
help if true, the help text will be displayed when the function exits.

Returns:

the sequence source, or null to indicate a failure
